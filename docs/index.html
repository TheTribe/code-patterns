    Documentation

    Code Patterns documentation is divided up into two parts: API documentation and code samples.

    API Documentation

    Code Patterns' API docs are generated by the Patterns Docs Generator, which uses a combination of Reflection and LINQ-to-XML.

    • Patterns.Framework
      blends Patterns with the .NET Framework
