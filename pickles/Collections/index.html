Collections

  • AddRange for Collections - As a developer, I want to be able to add a range of items to any collection using the same easy syntax that the generic List class provides.
  • Each for Enumerables - As a developer, I want to be able to easily run a delegate, method group, or lambda expression across a set of items, while optionally setting the actions to run in parallel.
